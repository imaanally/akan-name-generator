# Akan Name Generator

## Description
Akan Name Generator is a simple web app that helps users find their Akan name based on their birthdate and gender.

## Author
Imaan Ali

## Live Site
https://imaanally.github.io/akan-name-generator/

## Setup Instructions
1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## Technologies Used
- HTML
- CSS
- JavaScript

## BDD
| Input | Behavior | Output |
|---|---|---|
| Valid birthdate and gender | Calculates day of birth | Displays Akan name |
| Empty form | Shows alert | User fills required fields |
| Invalid day | Shows alert | User enters day between 1 and 31 |
| Invalid month | Shows alert | User enters month between 1 and 12 |
